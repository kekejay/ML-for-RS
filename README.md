**Requirements**

To install the requirements, follow these steps:

1. Ensure you are using Python 3.10.0
   
2. Run the following command to install the required packages:

     pip install -r requirements.txt

**Data Preparation**

Prepare a dataset containing Raman shifts and corresponding labels, preferably in a csv table

**Training**

You can provide your own data and train the model according to the example.ipynb file
